# Ứng dụng Dự báo Rủi ro Khách hàng (Mô hình 5C) 🏦

Ứng dụng web Streamlit này được chuyển đổi từ notebook Machine Learning đánh giá rủi ro khách hàng. Cốt lõi của ứng dụng sử dụng thuật toán **Logistic Regression** để học và đưa ra dự báo khả năng vi phạm/có rủi ro (`PD` - Probability of Default) dựa trên các thang đo thuộc 5 phương diện (Mô hình 5C): Tài chính (TC), Năng lực (NL), Điều kiện (DK), Vốn (V), và Tài sản (TS).
